# Exercise 0.6: New note in single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User writes note and clicks Save
    Note right of browser: JS handles submit and calls preventDefault()
    Note right of browser: Browser updates local notes list and redraws UI

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa\nContent-Type: application/json\nBody: { content, date }
    activate server
    Note left of server: Server stores the new note
    server-->>browser: 201 Created
    deactivate server

    Note right of browser: No page reload and no extra GET requests
```
